# 🎤 Talks — DTD USM
 
Repositorio de presentaciones y charlas realizadas por la **Dirección de Transformación Digital (DTD)** de la **Universidad Técnica Federico Santa María (USM)**.
 
Las temáticas abarcan transformación digital, analítica de datos, inteligencia artificial y gestión del cambio tecnológico en el contexto universitario.
 
> Más sobre la DTD: [transformaciondigital.usm.cl/dtd](https://transformaciondigital.usm.cl/dtd/)
