Los tres estados de Git

Staging Area

Repository

El ciclo básico


# Ver qué ha cambiado
git status
git diff

# Preparar cambios (staging)
git add archivo.py          
git add .                  

# Guardar snapshot (commit)
git commit -m "feat: agregar función de predicción"

# Sincronizar con GitHub
git push origin main
git pull origin main
